Git Commands

Command Description
git init To initialize the repo
git log Shows previous commits of the repo
git stausShows current tracking status of the files
git clone "repoUrl"To clone the remote repo to local
git add . To stage all the files that are changed
git add filename To stage the specific file
git commit -m msg To commit the staged files in local
git push To push the the commits to remote repo
git pull To pull the commits from the remote repo to local
git reset --hard commitId To reset the repo till commit id with deleting the later commits code
git reset --soft commitId To reset the repo till committ id with deleting the later commits code
git checkout branchname To switch to a different branch
git config --global init.defaultBranch main To change the default branch name when git init is used
git remote add origin "remoteURL" To add the origin repo URL for the local repo
git push --set-upstream origin "branchName" To set the upstream branch for the local branch
